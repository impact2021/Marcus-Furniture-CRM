# Home Shield Enquiries CRM

A WordPress plugin for managing painter enquiries with contact form, admin dashboard, and automated quote generation.

## Overview

This plugin provides a complete CRM solution for Home Shield Painters, including:

- Public contact form with NZ address autocomplete
- Admin dashboard for managing enquiries
- Status tracking (Not Actioned, Emailed, Quoted, Completed, Dead)
- Email system with editable quote tables
- Automatic GST calculation (15%)

## Quick Start

1. Upload the plugin files to `/wp-content/plugins/home-shield-crm/`
2. Activate the plugin in WordPress admin
3. Configure Google Maps API key and timezone in Settings
4. Add `[hs_contact_form]` shortcode to any page

For detailed installation and usage instructions, see [PLUGIN_README.md](PLUGIN_README.md)

## Features

✅ Contact form with 5 pre-defined job types  
✅ NZ-only address autocomplete (Google Places API)  
✅ Admin dashboard with filtering tabs  
✅ Status change tracking  
✅ Professional quote emails  
✅ Editable quote table with automatic GST calculation  
✅ Configurable timezone settings for accurate date/time display  
✅ Responsive design  

## Requirements

- WordPress 5.0+
- PHP 7.2+
- Google Maps API key
